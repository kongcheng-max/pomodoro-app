## 二、白噪声 ✅ 通过

### 2.1 播放控制

- [x] 13. 番茄钟区域底部出现音量图标按钮
- [x] 14. 点击音量图标 → 弹出白噪声面板
- [x] 15. 面板标题"白噪音"，有关闭按钮（X）
- [x] 16. 点击"开启"按钮 → 开始播放（按钮变为红色"关闭"）
- [x] 17. 听到持续的背景噪声（沙沙声）

### 2.2 类型切换

- [x] 18. 点击"白噪音""粉红噪音""棕色噪音" → 音色有可感知变化
- [x] 19. 切换回"白噪音" → 恢复初始音色

### 2.3 音量控制

- [x] 20. 拖动音量滑块到最左 → 静音或极低
- [x] 21. 拖动到最右 → 音量明显增大
- [x] 22. 拖动到中间位置 → 音量适中

### 2.4 持久化

- [x] 23. 选择"粉红噪音" + 音量 0.5
- [x] 24. 刷新页面 → 重新打开面板，类型=粉红噪音，音量=0.5
- [x] 25. 关闭播放后刷新 → 状态保持关闭

### 2.5 面板关闭

- [x] 26. 点击面板外空白区域 → 面板关闭
- [x] 27. 面板关闭时播放不中断
