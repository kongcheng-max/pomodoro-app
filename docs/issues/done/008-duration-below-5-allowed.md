# 自定义时长允许输入低于 5 分钟的值

## 复现步骤

1. 打开番茄钟设置弹窗
2. 在自定义输入框中输入 `1`
3. 按 Enter

## 预期行为

按 `feature-spec.md` §1.2，专注时长可自定义范围为 **5 ~ 120 分钟**，输入 1 应被拒绝。

## 实际行为

`TimerSettings.tsx` 自定义输入 `min={5}`（第 87 行），`handleCustom` 中 `val >= 5` 检查（第 23 行）。但 HTML `min` 属性在某些浏览器上不阻止手动输入 `1`，仅表单提交时触发验证提示——用户体验不佳且提示不明显。

虽然 `handleCustom` 有 `val >= 5` 守卫，但用户输入 1 按 Enter 后无任何反馈（既不设置也不提示错误），造成困惑。

## 建议修复

- 输入 < 5 时给出视觉反馈（输入框红框 + 抖动动画，类似空标题处理）
- 或将下限在输入框旁以 placeholder 形式提示：`5~120 分钟`

## 环境信息

- 浏览器：不限
