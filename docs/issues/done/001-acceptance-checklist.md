# BUG-001 修复验收清单

> 修复项：TimerSettings 弹窗标题根据模式动态显示
> 验收人：产品部（手动）

---

## 复现检查

按以下步骤逐一验证，每步通过打 ✅

### 专注模式

- [ ] 1. 打开应用，确保当前为**专注模式**（标签显示"专注"）
- [ ] 2. 点击番茄钟下方的齿轮图标 ⚙️，打开设置弹窗
- [ ] 3. **确认弹窗标题显示"专注时长"**

### 短休息模式

- [ ] 4. 退出弹窗，切换到**短休息模式**
- [ ] 5. 再次点击齿轮图标，打开设置弹窗
- [ ] 6. **确认弹窗标题显示"短休息时长"**

### 长休息模式

- [ ] 7. 退出弹窗，切换到**长休息模式**
- [ ] 8. 再次点击齿轮图标，打开设置弹窗
- [ ] 9. **确认弹窗标题显示"长休息时长"**

### 切回专注模式

- [ ] 10. 退出弹窗，切回**专注模式**
- [ ] 11. 再次打开设置弹窗
- [ ] 12. **确认标题恢复显示"专注时长"**（无残留）

---

## 达标标准

- 三项模式标题全部正确
- 切换模式后即时生效，无需刷新
- 预设按钮、自定义输入框功能不受影响

---

> 全部 ✅ 即验收通过
