## 一、番茄钟增强

### 1.1 模式切换 UI

- [ ] 1. 打开应用，番茄钟上方出现 `专注` `短休息` `长休息` 三标签
- [ ] 2. 默认选中"专注"，白底红字高亮
- [ ] 3. 每个标签下方显示分钟数（25min / 5min / 15min）
- [ ] 4. 点击"短休息"→ 切换成功，高亮移动
- [ ] 5. 点击"长休息"→ 切换成功，高亮移动
- [ ] 6. 切回"专注"→ 正常
- [ ] 7. 计时运行中，点击其他标签 → **不可点击**（灰色 + 禁止光标）
- [ ] 8. 暂停状态同样锁定
- [ ] 9. 切换到短休息 → 齿轮按钮旁显示 `5 分钟`
- [ ] 10. 切换到长休息 → 显示 `15 分钟`
- [ ] 11. 各模式下打开设置弹窗 → 标题匹配（专注时长/短休息时长/长休息时长）

### 1.2 白噪声

- [ ] 12. 番茄钟下方出现音量图标按钮（齿轮旁）
- [ ] 13. 点击音量图标 → 弹出白噪声面板
- [ ] 14. 点击"开启"按钮 → 听到持续噪声（按钮变红色"关闭"）
- [ ] 15. 切换"白噪音""粉红噪音""棕色噪音" → 音色有可感知变化
- [ ] 16. 拖动音量滑块 → 音量实时变化
- [ ] 17. 选择粉红噪音 + 音量 0.5 → 刷新页面 → 打开面板确认状态已保存
- [ ] 18. 关闭播放后刷新 → 状态保持关闭
- [ ] 19. 点击面板外空白区域 → 面板关闭但不中断播放

### 1.3 自动切换

- [ ] 20. 将专注时长调至 **1 分钟**，切换到专注模式
- [ ] 21. 开始计时，等待归零
- [ ] 22. 通知弹出`"🍅 专注完成！"`
- [ ] 23. **1.5 秒后自动进入短休息**，计时器开始倒计时
- [ ] 24. 短休息归零 → 通知弹出`"☕ 休息时间到！"` → 1.5 秒后自动切回专注
- [ ] 25. 将长休息间隔调至 **2 轮**，完成 2 轮专注后 → 长休息触发
- [ ] 26. 自动切换时 `sessionCount` 正确递增

### 1.4 模式感知细节

- [ ] 27. 专注模式下按钮显示"开始专注"
- [ ] 28. 切换到短休息 → idle 按钮是否跟随变化（已知 BUG-005）
